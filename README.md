# Portfolio – Géomaticien / Analyste SIG

## Présentation

Géomaticien spécialisé en analyse spatiale appliquée aux énergies renouvelables (éolien, solaire, agrivoltaïsme). Ce portfolio présente mes projets et méthodologies SIG réalisés sous QGIS, avec automatisation Python (GeoPandas).

## Contenu du portfolio

- **Méthodologie SIG multicritère** – Identification des zones propices aux ENR
- **Identification de contraintes** – Zones réglementaires, environnementales et techniques
- **Filière éolienne** – Analyse spatiale et cartographie des servitudes (radars, aviation, zones protégées)
- **Modéleur graphique** – Automatisation des traitements éolien, PV et AgriPV sous QGIS
- **Filière PV et AgriPV** – Cartographie des zones compatibles

## Compétences

- QGIS – Analyse spatiale, cartographie thématique
- Python – GeoPandas, automatisation des traitements SIG
- Géoréférencement, détection de réseaux
- Modéleur graphique QGIS
